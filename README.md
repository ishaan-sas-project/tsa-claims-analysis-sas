This project analyzes Transportation Security Administration (TSA) claims data from 2002 to 2017 using Base SAS.
The objective is to clean, validate, analyze, and visualize claim patterns related to passenger property loss, personal injury, and related incidents across U.S. airports.

The analysis demonstrates data preparation, quality checks, reporting, and visualization using SAS procedures and ODS.
