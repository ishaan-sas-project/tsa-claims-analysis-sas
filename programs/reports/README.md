# TSA Claims Analysis – Report Output

This folder contains the PDF report generated by the
`tsa_claims_analysis.sas` program.

## Report Details
- File: claims_report.pdf
- Format: ODS PDF
- Style: Pearl
- Coverage Period: 2002–2017

## Description
The report summarizes TSA claims data through tables and visualizations,
including overall claim trends, state-level claim distributions, and
close amount statistics.

## Notes
- The report is generated automatically by the SAS program.
- To reproduce the report, update file paths and macro variables
  in `tsa_claims_analysis.sas` and rerun the program.
